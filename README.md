## Project Description
This project demonstrates JavaScript fundamentals including variables, functions, control flow, arrays, and objects. It includes a calculator and a student grade tracker.

## Technologies Used
- HTML5
- JavaScript

## Features
- Interactive calculator supporting +, -, *, /, %, **
- Student grade tracker with averages and reports
- Daily challenges (FizzBuzz, palindrome, etc.)

## How to Run
1. Clone the repository
2. Open `index.html` in your browser
3. Open browser console to view results

## Lessons Learned
- How to write JavaScript functions
- How to use arrays and objects
- How to manipulate data using map, filter, and reduce

## Challenges Faced
- Understanding how objects work
- Handling division by zero in calculator
- Solved by practicing and debugging step-by-step
